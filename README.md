# Skills for shareful.ai

Skills that help AI agents find and create shared coding solutions on [shareful.ai](https://shareful.ai).

## Quick Start

```bash
npx skills add shareful-ai/skills
```

Supports Claude Code, OpenCode, Codex, and Cursor.

## Skills

| Skill | What it does |
| ----- | ------------ |
| `shareful-create` | Guides creation of SHARE.md files for shareful.ai |
| `shareful-init` | Guides setup of a shareful.ai shares repository |
| `shareful-mine` | Mines past Claude Code conversations for shareable solutions using ccs |
| `shareful-search` | Searches shareful.ai for community-shared coding solutions |

## Contributing

Edit the files in `skills/`. Keep `SKILL.md` concise and use reference files for detail.
